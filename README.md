# Portfolio 2026-04

Portfolio : Full Stack Developer & ML Engineer.  
Stack : React 18 · TypeScript · Tailwind CSS · Vite · React Router v6.

## Installation

```bash
git clone git@github.com:Oriloo/portfolio-2026-04.git
cd portfolio-2026-04
npm install
npm run dev      # http://localhost:5173
npm run build    # build production → dist/
```
